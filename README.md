Stellar: Enhanced 3D Solar System Simulator

一个使用Three.js构建的交互式3D太阳系模拟器。

1st. 功能特点

- 逼真的太阳系3D模拟
- 交互式控制（旋转、缩放、平移）
- 行星信息面板
- 时间控制（加速、减速、暂停）
- 背景音乐
- 响应式设计

2nd. 部署

ONE 选项1: Netlify (推荐)

1. 将项目上传到GitHub仓库
2. 注册Netlify账户
3. 连接GitHub仓库
4. 自动部署

TWO 选项2: 传统Web服务器

将项目文件上传到支持HTTPS的Web服务器。

3rd. 安全特性

- 内容安全策略(CSP)
- XSS保护
- 点击劫持保护
- 安全的资源加载

4th. 浏览器支持

- Chrome 70+
- Firefox 65+
- Safari 12+
- Edge 79+
